# carbonemission-Azure Energy Emissions Lakehouse
 
CarbonLedger is a production-style Azure Data Engineering project designed to simulate enterprise carbon emissions and carbon intensity analytics across energy assets.
 
## Project Goal
 
The goal of this project is to build an end-to-end Azure Lakehouse using:
 
- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- PySpark
- Delta Lake
- SCD Type 2
- Delta MERGE
- Partitioning
- Spark Performance Tuning
- Gold Layer KPI Tables
 
### Architecture
 
Data flows through Medallion Architecture:
 
Bronze → Silver → Gold
